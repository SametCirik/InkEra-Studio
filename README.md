# InkEra - Simple Drawing Application

This is a personal drawing/painting application named **InkEra**, developed entirely using the **Java** programming language.

This project focuses on building a functional Graphical User Interface (GUI) from scratch, mastering event handling, and implementing core graphics rendering logic to create a simple, robust drawing experience.

---

## Features

* **Basic Drawing Tools:** Implements fundamental tools like **Pen/Pencil** and **Eraser**.
* **Color Selection:** Allows users to choose a color for drawing.
* **Canvas Interaction:** Supports drawing by clicking and dragging the mouse on the canvas.
* **Java-Based Desktop App:** Built using standard **Java** libraries (Swing/AWT) for desktop environment stability.

---

## Status and Planned Features (v1.0 Pre-Alpha)

**Current Status:** The core drawing logic and basic GUI structure are complete. This **Pre-Alpha** version is a foundational build focusing on stability.

### What's Next? (Future Plans)

* **Shape Drawing:** Adding tools for drawing basic shapes (Rectangle, Circle, Line).
* **Saving/Loading:** Implementing functionality to save the drawing as an image (e.g., PNG) and load previous drawings.
* **Brush Size Control:** Allowing the user to adjust the size/thickness of the brush.

---

## Development Environment and Technologies

* **Language:** Java
* **Framework/Library:** Standard Java Development Kit (JDK) and Swing/AWT (for the GUI) were used.
* **Development Environment:** IntelliJ IDEA / Eclipse
